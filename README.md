# c62yang_sidequest_w9

Project Title GBDA302 Week 9 Example 3: Debug Screen

Authors Karen Cochrane and David Han 
Editor: Cynthia Yang (21070916)

Description:
This project builds on a side scrolling platformer by adding a debug system to support testing and development. The game allows the player to move, jump, and attack while interacting with a tile based environment. A debug panel was implemented to toggle moon gravity, visualize hitboxes, and reset the player position, making it easier to test physics and collisions during gameplay.

Keys:
A / Left Arrow 
D / Right Arrow to move the player
W / Up Arrow to jump
Press Space to attack

For debug features:
Press G to toggle moon gravity on and off
H to show or hide hitboxes
R to reset the player back to the starting position

Learning Goals Learning Goals:
- Understand how to implement and modify physics systems such as gravity in a game environment
- Learn how to create a debug interface to support testing and development
- Practice using conditionals and key inputs to toggle game features dynamically
- Improve understanding of collision detection through hitbox visualization
- Gain experience structuring interactive systems in p5.js and p5play
